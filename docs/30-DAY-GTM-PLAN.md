## 💡 PRO TIPS

### Outreach
```
✓ Personalize every message (mention specific events)
✓ Lead with value, not features
✓ Use video when possible (Loom)
✓ Follow up 3-5 times (80% close on 5th touch)
✓ Ask for referrals from every conversation
```

### Demos
```
✓ Qualify before demo (budget, timeline, decision maker)
✓ Customize demo to their use case
✓ Show, don't tell
✓ Handle objections with questions
✓ Always have clear next step
```

### Closing
```
✓ Create urgency (limited spots, special pricing)
✓ Make it easy (simple contract, fast payment)
✓ Reduce risk (money-back guarantee, pilot option)
✓ Stay in touch (daily check-ins on active deals)
✓ Celebrate and ask for referrals
```
