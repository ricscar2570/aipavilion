## 2️⃣ FIRST CUSTOMER PLAYBOOK

### Discovery Call Script (30 min)

**Opening (5 min):**
```
"Thanks for taking the time [Name]. Before I show you 
the platform, I'd love to understand your current situation 
better. Tell me about [Company]'s virtual event experience..."

Questions to Ask:
1. What virtual events have you run before?
2. What platform did you use? What worked/didn't work?
3. What's your biggest challenge with virtual events?
4. How many attendees/exhibitors typically?
5. What's your timeline for next event?
6. Who makes the final decision?
7. What's your budget range?

Goal: Understand needs, qualify opportunity
```

**Demo (15 min):**
```
"Based on what you shared, let me show you specifically 
how AI Pavilion solves [their pain point]..."

Demo Flow:
1. Homepage (30 sec) - "This is what attendees see..."
2. Stand Detail (2 min) - Show AR demo on YOUR phone
3. 360° Tour (2 min) - Immersive walkthrough
4. Admin Dashboard (3 min) - "This is YOUR view..."
5. Analytics (2 min) - Real-time metrics
6. Deployment (2 min) - "Live in 10 minutes..."
7. Q&A (3 min)

Focus on THEIR use case, not all features
```

**Closing (10 min):**
```
"So [Name], based on what you've seen, does this solve 
[their specific problem]?"

Next Steps:
☐ Gauge interest (1-10 scale)
☐ Address any concerns
☐ Propose pilot program
☐ Schedule follow-up
☐ Send proposal

"Great! I'll send you a custom proposal today. 
 Let's schedule 30 min next [Day] to walk through it?"

"Sounds like you need to discuss internally. 
 Can I send some materials for your team? 
 Let's connect [Day] to discuss?"

If COLD:
"I understand timing isn't right. Can I check back 
 in [timeframe]? Any other stakeholders I should connect with?"

## 3️⃣ OBJECTION HANDLING SCRIPTS

### Price Objection

**"It's too expensive"**
```
Response:
"I understand budget is important. Let me put this 
in perspective...

Traditional platform: €5k-10k PER EVENT
AI Pavilion: €[X]k one-time, unlimited use

After just 2 events, you're saving money. Plus, 
you OWN this - no recurring fees, no lock-in.

What if we structured this differently? 
- Pilot for 1 event first (€[Y]k)
- Payment plan (3 installments)
- Revenue share model

What works better for your budget?"

KEY: Reframe as investment, show ROI, offer flexibility
```

**"We don't have budget right now"**
```
Response:
"I totally understand. When does your next budget 
cycle start?

In the meantime, can we:
- Run a free pilot for your next event?
- Start with a smaller scope?
- Find budget in a different department?

Also, what if this CREATES budget? If we can show 
ROI from increased exhibitor/attendee revenue, 
could that unlock funds?"

KEY: Future date commitment, creative solutions
```

### Technical Objection

**"Our team isn't technical enough"**
```
Response:
"That's actually a STRENGTH of AI Pavilion - 
it's designed for non-technical teams.

Here's what we handle:
✓ Complete deployment (we do it)
✓ All technical configuration
✓ 2 full training sessions
✓ Documentation in plain English
✓ 30 days support (we're always available)

After setup, managing it is like using WordPress - 
just click, upload, publish. No coding needed.

Plus, everything runs on AWS (99.9% uptime), 
so you don't need DevOps.

Want to see how simple the admin panel is?"

KEY: Emphasize support, show admin UI, compare to familiar tools
```

**"We're already using [Competitor]"**
```
Response:
"[Competitor] is a solid platform. What's working 
well for you there? And what could be better?

[Listen]

Based on what you shared, here's what makes 
AI Pavilion different:

vs [Competitor]:
✓ AR visualization (they don't have)
✓ 360° tours (they have basic 3D)
✓ You own the platform (vs subscription)
✓ €20/month (vs €500+/month)
✓ Gaming-specific features

We're not saying switch everything - but what 
if we run your next event on AI Pavilion? See 
the difference firsthand, no commitment?"

KEY: Respect competitor, highlight unique value, pilot offer
```

### Timing Objection

**"Not right now / Bad timing"**
```
Response:
"I appreciate the honesty. Help me understand - 
is it:
a) Timing within the year? (when's better?)
b) Need to see a pilot first?
c) Other priorities?

[Listen]

What if we:
- Schedule implementation for [better time]?
- Start with a small pilot now?
- Just set up a demo account you can explore?

Also - early bird pricing ends [Date]. If you 
decide later, it'll be 20% more. Can we at least 
lock in current pricing?"

KEY: Understand real objection, offer alternatives, create urgency
```

### Authority Objection

**"I need to run this by [Decision Maker]"**
```
Response:
"Absolutely - who else needs to be involved?

[Get names, titles, concerns]

Great! To help you present this internally, 
I can:
- Join the meeting (15 min presentation)
- Send materials tailored for [Decision Maker]
- Create a custom ROI analysis
- Provide references they can call

What would make it easiest for you to get 
buy-in?"

KEY: Become coach, help them sell internally, offer to join
```

### Risk Objection

**"What if it doesn't work?"**
```
Response:
"That's a smart concern. Here's how we mitigate risk:

1. 7-Day Satisfaction Guarantee
   If you're not happy 7 days after launch, 
   full refund. No questions.

2. Proven Track Record
   [X] successful events deployed
   [Y]% customer satisfaction
   [Testimonials if available]

3. Pilot Option
   Test with 1 event before full commitment

4. AWS Infrastructure
   99.9% uptime SLA - enterprise-grade

Honestly, biggest risk is NOT innovating while 
your competitors are. AR and 360° are becoming 
expected features in virtual events.

Want to start with a pilot to remove risk completely?"

KEY: Address directly, provide guarantees, reframe risk
